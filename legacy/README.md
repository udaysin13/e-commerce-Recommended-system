# Legacy Archive

This folder preserves the previous implementation and generated project notes.

## Contents

```text
legacy/
  backend/              Previous backend implementation
  frontend/             Previous frontend implementation
  docs/                 Previous docs and screenshots
  root-docs/            Previous root-level guides and config files
  README.md
```

## Usage

Use this archive as reference material only. New development should happen in the root-level `frontend/`, `backend/`, `recommendation-service/`, and `docs/` folders.

Do not add new application code here unless you are intentionally preserving old work.
